# Seller Circle FBA — Website

Premium landing page for the Seller Circle Amazon FBA course.

## 🚀 Live Demo
[https://seller-circle.vercel.app](https://seller-circle.vercel.app)

## 📁 Structure
```
/
├── index.html          # Main landing page
├── script.js           # Centralized JS + links system
├── vercel.json         # Vercel deployment config
├── sitemap.xml         # SEO sitemap
├── robots.txt          # SEO robots
├── pages/
│   ├── register.html   # Registration page
│   ├── checkout.html   # Checkout (Vodafone Cash / Instapay)
│   ├── login.html      # Login page
│   ├── dashboard.html  # Student dashboard
│   ├── course.html     # Course player
│   ├── thank-you.html  # Thank you page
│   ├── privacy.html    # Privacy policy
│   ├── terms.html      # Terms & conditions
│   └── refund.html     # Refund policy
└── assets/
    ├── images/
    └── results/        # Amazon sales screenshots
        ├── egypt/
        ├── saudi/
        └── ads/
```

## 🛠 Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- No frameworks, no dependencies
- Deployed on Vercel

## 📱 Features
- RTL Arabic design
- Dark premium theme (Amazon style)
- Mobile responsive
- SEO optimized
- Scroll animations
- Animated counters
- FAQ accordion
- Results gallery with lightbox
- Countdown timer
- WhatsApp checkout flow

## 🏃 Deploy
